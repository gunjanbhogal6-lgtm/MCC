# MCC

Welcome to the MCC repository! This is an Astro-based project.

## About

MCC is a modern web project built with Astro, providing a foundation for building fast, scalable web applications.

## Features

- Built with [Astro](https://astro.build/)
- Modern web development practices
- Optimized performance

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
    ```bash
    git clone https://github.com/gunjanbhogal6-lgtm/MCC.git
    cd MCC
    ```

2. Install dependencies
    ```bash
    npm install
    ```

3. Start the development server
    ```bash
    npm run dev
    ```

## Development

To run the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:3000`

## Building

To build for production:
```bash
npm run build
```
